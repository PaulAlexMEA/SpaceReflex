Ouverture du projet : 
- git clone https://github.com/PaulAlexMEA/SpaceReflex.git
- cd SpaceReflex
- npm install
- code .

Installation de Node JS et SGLite: 
- npm init -y
- npm install express
- npm install sqlite3

Framework utilisé : 
- Phaser pour la gestion d'affichage du jeu
- SQLite pour gérer le back plus simplement
- Opengameart pour les assets
